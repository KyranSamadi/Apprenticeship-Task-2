Development

Stage 4

This is the stage where the design is brought into real-world software. It is crucial that the development team write clean, optimised code in order to minimise the chance of errors or making the user experience sub-optimal. This process will involve both back-end and front-end development.

Inputs - Designs, best coding practices, technical documents

Outputs - Fully functional software, staging environment, backend setup, Content management setup

People Involved - Frontend developers, backend developers, project manager
