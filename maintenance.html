Maintenance

Stage 7

Post deployment, maintenance will be done on the software to ensure it stays live and functioning as intended with no issues. This could include fixing bugs, improving performance, adding new content or features or updating plugins for example. This process also ensures that the security of the software remains intact, by downloading security patches.

Inputs - Analytics data, client feedback, error logs, system monitoring

Outputs - Bug fixes, enhancements, updates

People Involved - Developers, hosting providers, client
